# Sistema de Gestión de Boletos para Jubilados

Aplicación web interna para la Municipalidad de Oberá. Esta web está destinada exclusivamente a Administradores y Supervisores; los beneficiarios no acceden a esta interfaz.

## Roles

- Administrador: gestiona beneficiarios, altas, modificaciones, bajas, recargas y bloqueo de tarjetas.
- Supervisor: consulta dashboard, reportes, anomalías, auditorías e incidencias.

## Menú principal

Dashboard, Beneficiarios, Recargas, Anomalías y Reportes.

## Ejecutar

```bash
npm install
npm run dev
```

Credenciales de prueba:

- Administrador: `admin` / `admin123`
- Supervisor: `supervisor` / `super123`
